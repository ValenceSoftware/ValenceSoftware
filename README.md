# Valence Launcher

Valence Launcher lets you install several versions of Minecraft: Bedrock Edition (Windows 10/11) side-by-side. Test beta builds, preview releases, and stable versions without uninstalling and reinstalling the game each time.

**Repository:** [github.com/ValenceSoftware/Valence-Launcher](https://github.com/ValenceSoftware/Valence-Launcher)

## Disclaimer

This tool will **not** help you pirate the game. It requires a Microsoft account that owns Minecraft from the Microsoft Store.

## Prerequisites

* A Microsoft account connected to Microsoft Store which **owns Minecraft for Windows**
* **Administrator permissions** on your user account
* **Developer mode** enabled for app installation in Windows Settings
* For beta versions: **subscribe to the Minecraft Beta program** via Xbox Insider Hub
* [Microsoft Visual C++ Redistributable](https://aka.ms/vs/16/release/vc_redist.x64.exe)

## Setup

1. Download the latest release from the [Releases](https://github.com/ValenceSoftware/Valence-Launcher/releases) page
2. Unzip to a folder of your choice
3. Run `ValenceLauncher.exe`

## Compiling

You'll need Visual Studio with Windows 10 SDK 10.0.17763 and .NET Framework 4.8. Open `ValenceLauncher.sln` and build.

## FAQ

**Can I run multiple Minecraft instances at once?**

No. You can install multiple versions, but only one can run at a time.
